Archive/backup Git repositories

This is a simple web application that allows you to archive/backup Git repositories. (Only GitHub repositories are currently supported.)

The Git repository is sent to Gitea (a Git hosting service) and added as a mirror. The mirror is then accessible from the web. The mirror is automatically updated when new commits are pushed to the original repository.

For example, https://github.com/PyGithub/PyGithub
